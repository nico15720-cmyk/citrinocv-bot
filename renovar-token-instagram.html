🔑 Renovar Token de Instagram — Citrino Bot

El token de Instagram expiró el 22-Jun-2026. Seguí estos 3 pasos:

Paso 1 — Generar nuevo token en Graph API Explorer

1. Hacé clic en el botón de abajo (ya estás logueado en Meta)

2. En el explorador, arriba a la derecha: seleccioná tu app Citrino

3. Clic en "Generate Access Token"

4. Seleccioná la página de Facebook vinculada a Instagram de Citrino

5. Marcá estos permisos: instagram_basic, instagram_manage_messages, pages_messaging

6. Copiá el token generado — lo vas a necesitar en el Paso 2

🔗 Abrir Graph API Explorer

Paso 2 — Convertir a token de larga duración (no expira en 60 días)

Necesitás el App ID y App Secret de tu app de Meta, y el token del Paso 1.

Los encontrás en: Mis Apps → Configuración básica

Abrí esta URL en el navegador — te va a devolver el token de larga duración en JSON:

🔗 Abrir URL

Buscá el campo "access_token" en la respuesta — ese es tu nuevo token.

Paso 3 — Actualizar en Railway

1. Abrí Railway → tu proyecto delightful-mercy → Variables

2. Buscá INSTAGRAM_ACCESS_TOKEN

3. Pegá el token nuevo del Paso 2 → guardá → Railway deploya solo

🔗 Abrir Railway
⚠️ Acordate también de hacer doble clic en _commit_fixes.bat para deployar los fixes del código (echo loop de Instagram, nombre desde CRM, etc.)
